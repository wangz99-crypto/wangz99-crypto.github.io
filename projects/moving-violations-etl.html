☁️ Automated Cloud ETL Pipeline

Serverless AWS data ingestion architecture integrating Lambda, EventBridge, RDS, and Secrets Manager for automated relational data loading.

📌 System Overview

This project implements a production-style cloud ETL pipeline using AWS Lambda for automated ingestion of external data sources into a relational RDS database environment.

  • Automated Lambda-based ingestion from public APIs
  • Scheduled orchestration via Amazon EventBridge
  • Secure database credentials using AWS Secrets Manager
  • Relational schema deployed on Amazon RDS
  • Incremental loading of weather and violations data
