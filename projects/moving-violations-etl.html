Serverless Cloud Data Ingestion Pipeline

AWS Lambda–based automated ETL system with secure RDS storage and daily scheduling.

📖 Project Background

This project implements a fully automated cloud ETL pipeline integrating District of Columbia Moving Violations data with Daily Weather data into a MySQL relational database hosted on AWS RDS.

Rather than focusing on analytics, the objective is to build a secure and reproducible data infrastructure capable of supporting future analytical or decision-support applications. The pipeline integrates:

  • Python-based ETL (local full load + Lambda incremental updates)
  • Automated daily scheduling via EventBridge
  • Secure credential storage using AWS Secrets Manager
  • Relational schema deployment on AWS RDS
  • Duplicate-safe incremental ingestion logic

Primary data sources include DC Open Data Moving Violations (JSON API) and VisualCrossing Daily Weather datasets.
